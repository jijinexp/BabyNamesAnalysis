def data_wrangle_shape(population,group,description):
        data=health_data[(health_data['population'].isin([population])) & (health_data['group'].isin([group])) &
                          (health_data['description'].isin(description))]
        data=data.reset_index(drop=True)
        data=data.drop(['population','group'],axis=1)
        data=data.set_index(['description'])
        data.columns.names=['Year']
        data=data.T
        data=data.sort_index()
        return data

#function to clean all cells of unusual nuisance characters
def clean_cells(field):
    #checks if cells are  float or int and returns int or float
    if (type(field) == float) | (type(field)== int): 
        return field
    #check for string cells for regular expression using regex findall loads value into object s
    #object s is then converted to float value and returned to cell
    elif type(field)== str:
        field = field.split(',')[0]
        [s] = re.findall(r'\d+\.*\d*', field)
        return float(s)

Cleaning Mechanism:¶

  • After reading in file.All non standard missing values, were standardised to NaN.

  • Year column was parsed to convert to date format using pd.to_dataframe method.

  • Old index was dropped, year column set as index and sorted.

  • clean_cells function created to clean nuisance and unusual characters, which check if cell content is a string using regex pattern exclude all characters, except numbers and convert to float.It also checks if int or float return as it is.

  • function is called using applymap and implemented on nz_indicators dataset.

  • Used head() function to see if cleaning has happened.

After cleaning:¶

  • Data has been prepared for topic of choice by dropping columns not required using drop function.

  • Columns of interest relevant to each topic i.e for Housing and Labour is sliced so data is used only for previous 10 years from 2007 to 2017.This is because most columns have completed columns in terms of data.
